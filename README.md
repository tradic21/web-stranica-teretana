# Web stranica teretane

Timski web projekt: izrada web stranice za fiktivni fitness studio.

## Sadržaj
Stranica uključuje više podstranica (početna, treninzi, članstvo, posebna ponuda, o nama, kontakt) te slike i ikone korištene u dizajnu.

## Tehnologije
- HTML, CSS, JavaScript
- Bootstrap

## Pokretanje
Otvoriti `Index.html` u web pregledniku.

## Struktura
- `slike/` – slike korištene na stranicama  
- `ikone/` – ikone korištene u dizajnu
